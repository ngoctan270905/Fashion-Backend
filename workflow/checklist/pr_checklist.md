# PR Checklist

## Code
- [ ] Code đã tuân thủ coding_rules chưa?
- [ ] Không có code thừa không?
- [ ] Đặt tên biến/function rõ ràng chưa?

## Testing
- [ ] Đã viết test chưa?
- [ ] Coverage đạt 80% chưa?
- [ ] Test đã pass hết chưa?

## Review
- [ ] Reviewer đã approve chưa?
- [ ] Các góp ý đã được sửa chưa?

## Security
- [ ] Không có lỗ hổng bảo mật không?
- [ ] Input đã được validate chưa?
